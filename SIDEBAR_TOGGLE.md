# 侧边栏收起功能更新

## 🎯 更新内容

为左侧文件列表侧边栏添加了**可收起/展开**功能，让用户可以最大化视图区域。

## ✨ 功能特性

### 收起/展开按钮
- 📍 **位置**：侧边栏右侧边缘中间
- 🎨 **样式**：圆形按钮，带箭头图标
- ⏩ **收起**：点击后侧边栏收缩至 48px 宽度
- ⏪ **展开**：再次点击恢复 288px 宽度
- 🎭 **动画**：平滑过渡动画（300ms）

### 收起状态
当侧边栏收起时：
- 📤 **快速上传**：显示上传按钮图标
- 🔢 **文件计数**：显示已上传文件数量
- 👈 **展开提示**：箭头按钮提示可以展开

### 展开状态
侧边栏展开时显示完整功能：
- 📁 文件列表
- 📤 上传区域
- 📋 任务队列
- 🗑️ 删除按钮

## 🛠️ 技术实现

### 修改的组件

#### 1. `Sidebar.tsx` - 基础侧边栏组件
```typescript
interface SidebarProps {
  children: ReactNode;
  className?: string;
  collapsed?: boolean;      // 新增：收起状态
  onToggle?: () => void;    // 新增：切换回调
}
```

**特性**：
- 使用 CSS `transition-all duration-300` 实现平滑动画
- 收起时宽度从 `w-72` (288px) 变为 `w-12` (48px)
- 按钮使用绝对定位，居中显示在右侧边缘
- 箭头图标根据状态旋转 180 度

#### 2. `AppSidebar.tsx` - 应用侧边栏
```typescript
interface AppSidebarProps {
  // ... 原有 props
  collapsed?: boolean;      // 新增
  onToggle?: () => void;    // 新增
}
```

**特性**：
- 条件渲染内容（收起时隐藏内容）
- 传递状态到基础 Sidebar 组件

#### 3. `Index.tsx` - 主界面
```typescript
const [sidebarCollapsed, setSidebarCollapsed] = useState(false);
```

**新增功能**：
- 状态管理：`sidebarCollapsed`
- 快速上传栏：侧边栏收起时显示
- 文件计数显示

## 🚀 使用方法

1. **收起侧边栏**：
   - 点击侧边栏右侧边缘的圆形按钮
   - 或双击侧边栏边界（未来功能）

2. **展开侧边栏**：
   - 点击左侧窄条上的箭头按钮

3. **收起状态下上传**：
   - 点击 📤 图标
   - 选择 DXF 文件

## 💡 使用场景

- ✅ **查看大型图纸**：收起侧边栏获得更大视图空间
- ✅ **专注 3D 预览**：最大化显示区域
- ✅ **多显示器**：在有限空间内工作
- ✅ **快速上传**：无需展开即可上传文件
