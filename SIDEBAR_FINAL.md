## 🎯 按钮设计

### 外观
- **形状**：圆形（24px × 24px）
- **颜色**：主题色（accent）背景
- **边框**：2px 白色边框，与背景分离
- **阴影**：轻微阴影，浮于表面
- **图标**：SVG 箭头（可旋转）

### 位置
- **水平**：侧边栏右边缘，`-right-3` 使按钮居中于边界
- **垂直**：垂直居中，`top-1/2 -translate-y-1/2`
- **层级**：`z-10` 确保在最上层

### 交互
- **悬停**：放大 110% (`hover:scale-110`)
- **点击**：切换收起/展开状态
- **提示**：Tooltip 显示"收起侧边栏"或"展开侧边栏"

## ✨ 动画效果

### 收起动画
1. 侧边栏宽度从 288px 过渡到 0px
2. 按钮保持在原位置（绝对定位）
3. 箭头旋转 180°（从 `<` 到 `>`）
4. 时长：300ms
5. 缓动：ease-in-out

### 展开动画
1. 侧边栏宽度从 0px 过渡到 288px
2. 按钮保持在原位置
3. 箭头旋转 180°（从 `>` 到 `<`）
4. 时长：300ms
5. 缓动：ease-in-out

## 🛠️ 技术实现

### Sidebar.tsx
```tsx
<div className="flex relative">
  {/* 侧边栏主体 */}
  <aside className={`${collapsed ? 'w-0' : 'w-72'} ...`}>
    {!collapsed && children}
  </aside>
  
  {/* 按钮 - 绝对定位到右侧 */}
  {onToggle && (
    <button
      className="absolute -right-3 top-1/2 -translate-y-1/2 z-10 ..."
      onClick={onToggle}
    >
      <svg className={`transition-transform duration-300 ${collapsed ? 'rotate-180' : ''}`}>
        <path d="M15 19l-7-7 7-7" />
      </svg>
    </button>
  )}
</div>
```

### 关键 CSS 类
```css
/* 按钮定位 */
absolute -right-3 top-1/2 -translate-y-1/2 z-10

/* 按钮样式 */
w-6 h-6 rounded-full
bg-accent hover:bg-accent/90
border-2 border-background shadow-md

/* 交互动画 */
transition-all hover:scale-110

/* 箭头旋转 */
transition-transform duration-300 rotate-180
```

## 💡 设计优势

### 位置优势
✅ **分界清晰**：按钮位于两个区域中间，明确标识可调节边界  
✅ **不占空间**：跨边界居中，不占用侧边栏或主视图空间  
✅ **易于发现**：圆形浮起设计，视觉突出  
✅ **符合习惯**：类似常见 UI 的侧边栏折叠按钮位置  

### 交互优势
✅ **一键切换**：单击即可收起/展开  
✅ **视觉反馈**：悬停放大、箭头旋转  
✅ **状态明确**：箭头方向指示当前状态  
✅ **平滑动画**：300ms 过渡，流畅自然  

### 空间优势
✅ **收起后零占用**：侧边栏宽度变为 0，主视图最大化  
✅ **按钮常驻**：收起后按钮仍在原位，方便重新展开  
✅ **边界清晰**：侧边栏边框明确划分区域
