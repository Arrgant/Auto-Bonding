# 界面布局更新说明

## 📐 新布局设计

主界面现在采用**左右分屏**布局：

```
┌─────────────────────────────────────────────────────────────┐
│                      顶部导航栏                              │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│  侧边栏   │              主视图区域                          │
│          │  ┌──────────────┬──────────────┐                │
│  - 文件   │  │              │              │                │
│  - 任务   │  │   2D DXF     │    3D 预览    │                │
│          │  │   (左侧)     │    (右侧)     │                │
│          │  │              │              │                │
│          │  │              │              │                │
│          │  └──────────────┴──────────────┘                │
├──────────┴──────────────────────────────────────────────────┤
│                        状态栏                                │
└─────────────────────────────────────────────────────────────┘
```

## ✨ 主要功能

### 左侧面板 - 2D DXF 视图
- 📤 **DXF 文件导入**：支持拖拽或点击上传
- 🔍 **2D 图纸显示**：使用 Three.js 正交相机渲染 DXF 实体
- 📐 **支持的实体类型**：
  - LINE（直线）
  - CIRCLE（圆）
  - POLYLINE / LWPOLYLINE（多段线）
- 🎨 **自动缩放**：根据图纸内容自动调整视图范围
- 📊 **网格背景**：显示参考网格便于测量

### 右侧面板 - 3D 预览
- 🎯 **焊点坐标显示**：不同类型的焊点使用不同颜色
  - 🟡 球焊点 (ball) - 金色
  - 🔵 楔焊点 (wedge) - 蓝色
  - 🟢 缝焊点 (stitch) - 绿色
- 🔗 **线弧模拟**：使用金色管状几何体连接焊点
- 🖱️ **交互控制**：支持旋转、缩放、平移
- 📐 **坐标网格**：蓝色系网格和坐标轴

### 侧边栏改进
- 📁 **文件列表**：显示所有上传的 DXF 文件
- 👆 **点击选择**：点击文件可在左侧 2D 视图中显示
- 📊 **状态指示**：
  - 📄 待处理
  - ⏳ 转换中（带进度条）
  - ✅ 已完成
- 🗑️ **删除功能**：鼠标悬停显示删除按钮

### 标签页导航
- **视图**：左右分屏的 2D/3D 视图
- **参数配置**：转换参数设置
- **DRC 检查**：设计规则检查

## 🛠️ 技术实现

### 新增组件
- `DXF2DViewer.tsx` - DXF 2D 查看器组件
  - 使用 `dxf-parser` 解析 DXF 文件
  - Three.js OrthographicCamera 正交渲染
  - 支持多种 DXF 实体类型

### 依赖更新
```bash
npm install dxf-parser
```

### 修改的文件
- `Index.tsx` - 主界面布局重构
- `AppSidebar.tsx` - 添加文件选择功能
- `types/index.ts` - 类型定义（已有）

## 🚀 使用方法

1. **启动前端开发服务器**：
   ```bash
   cd Auto-Bonding/frontend
   npm run dev
   ```

2. **上传 DXF 文件**：
   - 拖拽 DXF 文件到主界面
   - 或点击侧边栏的上传区域

3. **查看 2D 图纸**：
   - 上传后自动在左侧显示
   - 点击侧边栏文件可切换显示

4. **生成 3D 预览**：
   - 点击顶部"转换"按钮
   - 等待转换完成后在右侧查看

5. **测试文件**：
   - 使用 `sample.dxf` 测试基本功能
   - 包含矩形边框和 4 个圆孔

## 📝 注意事项

1. **DXF 版本**：目前支持 AC1009 (AutoCAD R12) 格式的 DXF
2. **实体支持**：仅支持 LINE、CIRCLE、POLYLINE 等基本实体
3. **图层颜色**：支持 DXF 中定义的颜色，默认黑色
4. **性能**：大型 DXF 文件可能需要几秒解析时间
