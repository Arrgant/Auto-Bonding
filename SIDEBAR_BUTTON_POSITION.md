# 侧边栏按钮位置更新

## 📍 按钮位置调整

收起/展开按钮现在位于**文件列表左侧**，而不是侧边栏右边缘。

## 🎨 新布局

```
┌───┬──────────────┬─────────────────────────────────┐
│ < │ 📁 文件列表   │                                 │
│   │              │     主视图区域                   │
│ < │ 📤 上传       │  ┌──────────┬──────────┐       │
│   │              │  │          │          │       │
│ < │ 📄 file1.dxf │  │  2D DXF  │  3D 预览  │       │
│   │ 📄 file2.dxf │  │          │          │       │
│ < │ 📋 任务队列   │  └──────────┴──────────┘       │
└───┴──────────────┴─────────────────────────────────┘
  ↑
  收起/展开按钮列
```

## 🔄 两种状态

### 展开状态（默认）
```
┌───┬──────────────────┐
│ < │ 📁 文件列表        │
│   │                  │
│   │ 📤 点击上传 DXF 文件 │
│   │                  │
│   │ 📄 example.dxf   │
│   │ 📄 test.dxf      │
│   │                  │
│   │ 📋 任务队列        │
└───┴──────────────────┘
    ↑
    按钮在左侧，点击可收起
```

### 收起状态
```
┌───┬──────────────────┐
│ > │                  │
│ 📤 │   (隐藏)         │
│ 2  │                  │
│ 📄 │                  │
│ ⏳ │                  │
│   │                  │
└───┴──────────────────┘
    ↑
    按钮在左侧，点击可展开
    
显示内容：
- > 展开箭头
- 📤 快速上传
- 2  文件数量
- 📄 文件缩略图标
```

## ✨ 收起状态功能

在收起状态，左侧窄条显示：

| 元素 | 说明 |
|------|------|
| `>` | 展开箭头按钮 |
| 📤 | 快速上传按钮（点击选择文件） |
| `2` | 已上传文件数量 |
| 📄⏳✅ | 文件状态缩略图（最多显示 5 个） |

## 🎯 交互效果

### 悬停效果
- **按钮列**：悬停时背景色变浅
- **上传按钮**：悬停时放大
- **文件图标**：悬停时放大 + 高亮

### 点击效果
- **收起箭头**：侧边栏平滑收起
- **展开箭头**：侧边栏平滑展开
- **上传按钮**：打开文件选择对话框
- **文件图标**：切换到该文件（2D 视图）

## 📐 尺寸规格

```
展开状态：
├─ 按钮列：24px (w-6)
├─ 侧边栏：288px (w-72)
└─ 总计：312px

收起状态：
├─ 按钮列：24px (w-6)
├─ 侧边栏：0px (w-0)
└─ 总计：24px
```

## 🛠️ 技术实现

### Sidebar.tsx
```tsx
<div className="flex">
  {/* 左侧按钮列 */}
  <button className="w-6 ...">
    <svg>...</svg>
  </button>
  
  {/* 侧边栏主体 */}
  <aside className={`${collapsed ? 'w-0' : 'w-72'} ...`}>
    {!collapsed && children}
  </aside>
</div>
```

### AppSidebar.tsx
```tsx
{collapsed && (
  <div className="w-12 flex flex-col items-center py-4 gap-3">
    <label>📤 上传</label>
    <div>{files.length}</div>
    {files.slice(0, 5).map(...)}
  </div>
)}

{!collapsed && (
  <>
    {/* 完整文件列表 */}
  </>
)}
```

## 💡 优势

相比之前的右边缘按钮设计：

✅ **更符合直觉**：按钮在内容左侧，符合阅读习惯  
✅ **不遮挡视图**：按钮独立一列，不覆盖主视图  
✅ **节省空间**：收起后仅占 24px  
✅ **功能完整**：收起状态仍可上传和切换文件  
✅ **视觉清晰**：按钮和内容分离，层次分明  

## 🚀 使用提示

1. **快速收起**：点击左侧 `<` 箭头
2. **快速展开**：点击左侧 `>` 箭头
3. **快速上传**：收起状态下点击 📤
4. **切换文件**：收起状态下点击文件图标
