# 侧边栏按钮 - 竖条长条设计

## 📍 新按钮样式

收起/展开按钮从**圆形**改为**竖着的长条形**，更容易点击。

## 🎨 视觉效果

│ 📤 上传       │ ║ │                         │

│ 📄 file2     │ ║ │                         │
│              │ ║ │                         │
└──────────────┴───┴─────────────────────────┘
       侧边栏   ↑

## 📐 精确规格

### 尺寸
- **宽度**：8px (w-2)，悬停时 12px (hover:w-3)
- **高度**：96px (h-24)
- **位置**：跨边界居中，`-right-2`

### 样式
- **背景**：`bg-accent/80`（80% 透明度主题色）
- **悬停**：`hover:bg-accent`（100% 不透明）
- **形状**：左侧圆角 `rounded-l-lg`
- **边框**：上、下、左三边白色边框
- **阴影**：`shadow-lg` 长阴影

### 内容
```
┌─────────┐
│    <    │ ← 箭头图标（可旋转）
│    —    │ ← 装饰横线
└─────────┘
```

## ✨ 交互效果

### 悬停 (Hover)
- **宽度**：从 8px 增加到 12px
- **背景**：从 80% 不透明到 100%
- **光标**：pointer
- **提示**：Tooltip 显示"收起侧边栏"或"展开侧边栏"

### 点击 (Click)
- **收起**：侧边栏平滑收缩到 0px
- **展开**：侧边栏平滑展开到 288px
- **箭头**：旋转 180° 指示方向
- **动画**：300ms ease-in-out

## 🎯 设计优势

### 易点击性
✅ **更大的点击区域**：96px 高度 vs 之前 24px 圆形  
✅ **垂直覆盖广**：无论鼠标在哪个高度都能点到  
✅ **悬停反馈**：宽度增加提示可交互  
✅ **视觉突出**：竖条形状在边界上很明显  

### 美观性
✅ **简洁现代**：细长条设计符合现代 UI 趋势  
✅ **不突兀**：半透明背景，不遮挡视线  
✅ **方向明确**：箭头清晰指示收起/展开方向  
✅ **装饰细节**：底部横线增加设计感  

### 功能性
✅ **位置固定**：始终在侧边栏右边缘  
✅ **不占空间**：跨边界定位，不影响布局  
✅ **状态清晰**：箭头方向指示当前状态  
✅ **平滑过渡**：动画流畅自然  

## 🛠️ 技术实现

### Sidebar.tsx
```tsx
<button
  onClick={onToggle}
  className="
    absolute -right-2 top-1/2 -translate-y-1/2 z-10
    w-2 h-24
    bg-accent/80 hover:bg-accent
    rounded-l-lg
    flex flex-col items-center justify-center gap-1
    shadow-lg
    border-y-2 border-l-2 border-r-0 border-background
    transition-all hover:w-3
  "
>
  {/* 箭头图标 */}
  <svg className={`transition-transform duration-300 ${collapsed ? 'rotate-180' : ''}`}>
    <path d="M15 19l-7-7 7-7" />
  </svg>
  
  {/* 装饰横线 */}
  <div className="w-3 h-0.5 bg-accent-foreground/50 rounded-full"></div>
</button>
```

### CSS 类详解
```css
/* 定位 */
absolute -right-2 top-1/2 -translate-y-1/2 z-10

/* 尺寸 */

/* 颜色 */
bg-accent/80      /* 80% 透明度主题色 */

/* 形状 */
rounded-l-lg      /* 左侧圆角 */

/* 布局 */
flex flex-col items-center justify-center gap-1

border-y-2 border-l-2 border-r-0 border-background

/* 阴影 */
shadow-lg

/* 动画 */
transition-all hover:w-3
transition-transform duration-300
```

## 🎨 视觉细节

### 按钮结构
```
侧视图（放大）：
┌─────────────────┐
│                 │ ← 上边框（白色）
│        <        │ ← 箭头图标
│                 │
│        —        │ ← 装饰横线
│                 │
│                 │ ← 下边框（白色）
└─────────────────┘
    ↑
左边框（白色）   右侧开放（无边界）
```

### 颜色层次
```
背景色：bg-accent/80 (80% 透明度)
       ↓ hover
       bg-accent (100%)

边框：border-background (白色)
      上、下、左三边
      右侧开放

阴影：shadow-lg
      向右投射

内容：text-accent-foreground
      箭头和横线
```
