# IGBT 功率器件键合指南

本指南介绍如何使用 Auto-Bonding 处理 IGBT（绝缘栅双极晶体管）功率器件的键合图转换。

---

## 🔌 什么是 IGBT？

IGBT（Insulated Gate Bipolar Transistor）是一种高压大电流功率半导体器件，广泛应用于：

- 电动汽车逆变器
- 工业电机驱动
- 太阳能逆变器
- 不间断电源 (UPS)
- 高频焊接设备

与普通 IC 不同，IGBT 键合需要应对：
- **高电压**（600V-3300V+）→ 需要更大的电气间距
- **大电流**（10A-100A+）→ 需要更粗的铝线或铝带
- **温度循环**（-40°C~150°C）→ 需要考虑热膨胀匹配
- **机械应力** → 需要更严格的键合强度要求

## 📋 IGBT 键合设计规则

### 1. 电气间距（根据电压等级）

| 电压等级 | 范围 | 最小间距 | 应用场景 |
|---------|------|---------|---------|
| 低压 | <100V | 0.5mm | 低压 MOSFET |
| 中压 | 100-600V | 1.0mm | 通用车规 IGBT |
| 高压 | 600-1200V | 2.0mm | 工业 IGBT |
| 超高压 | >1200V | 3.0mm | 高铁/电网 IGBT |

### 2. 引线选择

#### 铝线（最常用）
| 线径 | 截面积 | 最大电流* | 典型应用 |
|------|--------|----------|---------|
| 100μm | 0.0079 mm² | 2.4A | 小功率 |
| 200μm | 0.031 mm² | 9.4A | 中功率 |
| 300μm | 0.071 mm² | 21A | 标准 IGBT |
| 400μm | 0.126 mm² | 38A | 大功率 |
| 500μm | 0.196 mm² | 59A | 超大功率 |

*按 300 A/mm² 电流密度计算，实际应用需 80% 降额

#### 铝带（大电流应用）
| 尺寸 (宽×厚) | 截面积 | 最大电流* | 典型应用 |
|-------------|--------|----------|---------|
| 500×50μm | 0.025 mm² | 10A | 小功率 |
| 1000×75μm | 0.075 mm² | 30A | 中功率 |
| 1500×100μm | 0.15 mm² | 60A | 标准 IGBT |
| 2000×125μm | 0.25 mm² | 100A | 大功率 |
| 3000×200μm | 0.60 mm² | 240A | 超大功率 |

*按 400 A/mm² 电流密度计算（铝带散热更好）

### 3. 弧高设计

弧高计算公式：
```
loop_height = k × √(span × wire_diameter)
```

| 模式 | 铝线系数 k | 说明 |
|------|----------|------|
| 标准 IC | 1.8 | 普通应用 |
| IGBT | 2.0 | 考虑热膨胀 |
| 车规级 | 2.4 | 额外 20% 补偿 |

**最小弧高限制**：IGBT 模式 ≥0.5mm（避免塌丝）

### 4. 焊盘尺寸要求

| 焊盘类型 | 最小尺寸 | 说明 |
|---------|---------|------|
| 发射极 (Emitter) | 0.3mm | 大电流输出 |
| 集电极 (Collector) | 0.5mm | 高压输入 |
| 栅极 (Gate) | 0.2mm | 控制信号 |

## 🛠️ 使用 Auto-Bonding 处理 IGBT

### 方法 1: Web 界面（推荐）

1. **选择模式**
   - 点击顶部模式切换按钮
   - 选择 `IGBT` 或 `车规级`

2. **配置参数**
   ```
   工作电压：600V（自动确定间距要求）
   引线类型：铝线
   线径：300μm（0.3mm）
   材料：铝线
   弧高系数：2.0（自动设置）
   ```

3. **上传 DXF 文件**
   - 拖拽文件到上传区域
   - 支持批量上传

4. **运行 DRC 检查**
   - 自动执行 IGBT 特定规则检查
   - 查看违规报告（错误/警告分类）

5. **导出结果**
   - STEP 文件：3D 模型预览
   - CSV/KS/ASM：打线坐标

### 方法 2: API 调用

```python
import requests

# IGBT 模式配置
config = {
    "mode": "igbt",
    "operating_voltage": 600,
    "wire_type": "al_wire",
    "default_wire_diameter": 0.3,
    "default_material": "aluminum",
    "loop_height_coefficient": 2.0,
    "export_format": "STEP"
}

# 上传文件
files = {"file": open("igbt_bonding.dxf", "rb")}
data = {"config": json.dumps(config)}

response = requests.post("http://localhost:8000/convert", files=files, data=data)
result = response.json()

print(f"转换成功：{result['message']}")
print(f"下载链接：{result['download_url']}")
```

### 方法 3: 命令行

```bash
# IGBT 模式转换
python cli.py convert igbt_bonding.dxf \
    --mode igbt \
    --voltage 600 \
    --wire-type al_wire \
    --diameter 0.3 \
    --material aluminum \
    --output igbt_model.step
```

## ✅ DRC 检查项目（IGBT 模式）

### 电气规则
- [ ] 电压间距检查（根据工作电压）
- [ ] 电流承载能力验证
- [ ] 网络短路检查

### 机械规则
- [ ] 引线跨度检查（最大 8mm 铝线 / 5mm 铝带）
- [ ] 弧高范围验证（0.5-3.0mm）
- [ ] 焊盘尺寸检查（发射极/集电极/栅极）

### 可靠性规则（车规级）
- [ ] 热膨胀补偿验证
- [ ] 疲劳寿命评估
- [ ] 温度循环耐受性

## 📊 典型应用案例

### 案例 1: 600V/50A IGBT 模块

**需求**：
- 工作电压：600V
- 最大电流：50A
- 封装：TO-247

**配置**：
```json
{
  "mode": "igbt",
  "operating_voltage": 600,
  "expected_current": 50,
  "wire_type": "al_ribbon",
  "ribbon_size": "1500×100μm",
  "min_spacing": 1.0
}
```

**结果**：
- ✅ 电流承载能力：60A > 50A（满足）
- ✅ 电气间距：1.2mm > 1.0mm（满足）
- ✅ 弧高：1.8mm（在 0.5-3.0mm 范围内）

### 案例 2: 1200V/100A 车规级 IGBT

**需求**：
- 工作电压：1200V
- 最大电流：100A
- 车规级：AEC-Q100
- 温度范围：-40°C ~ 150°C

**配置**：
```json
{
  "mode": "automotive",
  "operating_voltage": 1200,
  "expected_current": 100,
  "wire_type": "al_ribbon",
  "ribbon_size": "2000×125μm",
  "min_spacing": 2.0,
  "thermal_compensation": true
}
```

**结果**：
- ✅ 电流承载能力：100A（刚好满足，建议并联）
- ✅ 电气间距：2.2mm > 2.0mm（满足高压要求）
- ✅ 弧高：2.4mm（车规级补偿后）
- ✅ 疲劳寿命：满足 AEC-Q100 Grade 0

## ⚠️ 常见错误与解决方案

### 错误 1: 电流承载能力不足

**报错**：
```
❌ 电流承载能力不足：预期 60A > 最大 45A
```

**解决方案**：
1. 增加线径（如 300μm → 400μm）
2. 改用铝带（散热更好）
3. 多根引线并联

### 错误 2: 电压间距不足

**报错**：
```
❌ 电压间距不足 (600V): 实际 0.8mm < 要求 1.0mm
```

**解决方案**：
1. 调整焊盘布局，增加间距
2. 添加绝缘胶（点胶工艺）
3. 使用更高耐压的封装

### 错误 3: 引线跨度过大

**报错**：
```
❌ 引线跨度过大 (9.5mm)，需要中间支撑或改用铝带
```

**解决方案**：
1. 添加中间焊点（stitch bond）
2. 改用铝带（最大跨度 5mm）
3. 优化芯片布局，缩短跨度
