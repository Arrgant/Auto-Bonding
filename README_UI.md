# 🎨 Auto-Bonding 界面更新

## ✅ 已完成的改动

### 1. 左右分屏布局
主界面现在采用左右分屏设计：
- **左侧**：DXF 2D 视图（显示导入的二维图纸）
- **右侧**：3D 预览（根据二维图生成的三维焊点坐标）

### 2. DXF 2D 查看器
新建组件 `DXF2DViewer.tsx`，功能包括：
- 📤 支持 DXF 文件拖拽上传
- 🔍 解析并渲染 DXF 实体（LINE、CIRCLE、POLYLINE）
- 📐 使用 Three.js 正交相机进行 2D 渲染
- 🎨 自动缩放以适应图纸内容
- 📊 显示网格背景便于参考

### 3. 侧边栏增强
- 👆 点击文件可在 2D 视图中显示
- 🎯 高亮显示当前选中的文件
- 📊 文件状态指示（待处理/转换中/已完成）
- ⏪ **可收起/展开**：点击侧边栏边缘的箭头按钮可收起侧边栏，最大化视图区域

### 4. 依赖安装
```bash
npm install dxf-parser
```

## 📝 注意事项

1. **DXF 格式**：目前支持 AC1009 (AutoCAD R12) 格式
2. **实体类型**：支持 LINE、CIRCLE、POLYLINE
3. **演示数据**：3D 坐标目前是演示数据，真实转换需要后端支持
